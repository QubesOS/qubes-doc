qvm-remove
==========

NAME
----

qvm-remove - remove a VM

Date  
2012-04-11

SYNOPSIS
--------

qvm-remove [options] \<vm-name\>

OPTIONS
-------

-h, --help  
Show this help message and exit

-q, --quiet  
Be quiet

--just-db  
Remove only from the Qubes Xen DB, do not remove any files

--force-root  
Force to run, even with root privileges
