qvm-clone
=========

NAME
----

qvm-clone - clones an existing VM by copying all its disk files

Date  
2012-04-10

SYNOPSIS
--------

`qvm-clone [options] <src-name> <new-name>` 

OPTIONS
-------

`-h, --help`  
Show this help message and exit

`-q, --quiet`  
Be quiet

`-p DIR_PATH, --path=DIR_PATH`  
Specify path to the template directory
