```
===========
qubes-prefs
===========

NAME
====
qubes-prefs - display system-wide Qubes settings, such as:

- clock VM
- update VM
- default template
- default firewallVM
- default kernel
- default netVM

SYNOPSIS
========
| qubes-prefs
```
