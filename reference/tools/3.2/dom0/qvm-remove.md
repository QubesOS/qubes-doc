```
==========
qvm-remove
==========

NAME
====
qvm-remove - remove a VM

SYNOPSIS
========
| qvm-remove [options] <vm-name>

OPTIONS
=======
-h, --help
    Show this help message and exit
-q, --quiet
    Be quiet   
--just-db
    Remove only from qubes.xml; do not remove any files
--force-root
    Force to run, even with root privileges
```
