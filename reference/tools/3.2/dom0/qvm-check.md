qvm-check
=========

NAME
====
qvm-check - Specify no state options to check if VM exists

SYNOPSIS
========
| qvm-check [options] <vm-name>

OPTIONS
=======
-h, --help
    Show this help message and exit
-q, --quiet
    Be quiet
--running
    Determine if VM is running
--paused
    Determine if VM is paused
--template
    Determine if VM is a template
