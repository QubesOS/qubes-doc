qvm-copy-to-vm
==============

NAME
----

qvm-copy-to-vm - copy specified files to specified destination VM

Date  
2012-05-30

SYNOPSIS
--------

qvm-copy-to-vm [--without-progress] dest\_vmname file [file]+

OPTIONS
-------

--without-progress  
Don't display progress info
