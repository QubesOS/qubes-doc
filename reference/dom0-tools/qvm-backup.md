qvm-backup
==========

NAME
----

qvm-backup

Date  
2012-04-10

SYNOPSIS
--------

qvm-backup [options] \<backup-dir-path\>

OPTIONS
-------

-h, --help  
Show this help message and exit

-x EXCLUDE\_LIST, --exclude=EXCLUDE\_LIST  
Exclude the specified VM from backup (might be repeated)
